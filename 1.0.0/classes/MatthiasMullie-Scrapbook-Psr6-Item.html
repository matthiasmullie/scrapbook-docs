public setExpiration([mixed $ttl = null ]) : static
Parameters
$ttl : mixed = null
  • If an integer is passed, it is interpreted as the number of seconds after which the item MUST be considered expired.
  • If a DateTime object is passed, it is interpreted as the point in time after which the item MUST be considered expired.
  • If null is passed, a default value MAY be used. If none is set, the value should be stored permanently or for as long as the implementation allows.
